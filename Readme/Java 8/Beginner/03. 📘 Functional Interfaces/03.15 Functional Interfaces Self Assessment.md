# 3.15 Functional Interfaces Self Assessment

Use this checklist to evaluate your understanding of Functional Interfaces.

If you can confidently answer **Yes** to most of these questions without referring to your notes, you're well prepared for Java interviews.

---

# Foundation

- [ ] I can explain what a Functional Interface is.
- [ ] I know what a Single Abstract Method (SAM) is.
- [ ] I understand why Lambda Expressions require Functional Interfaces.
- [ ] I understand the concept of Passing Behavior as a Parameter.
- [ ] I know when to create a custom Functional Interface.

---

# Core Functional Interfaces

## Predicate

- [ ] I know when to use `Predicate`.
- [ ] I can write a `Predicate` using a Lambda Expression.
- [ ] I understand `test()`, `and()`, `or()`, and `negate()`.

---

## Function

- [ ] I know when to use `Function`.
- [ ] I understand `apply()`.
- [ ] I understand `compose()`.
- [ ] I understand `andThen()`.
- [ ] I understand `identity()`.

---

## Consumer

- [ ] I know when to use `Consumer`.
- [ ] I understand `accept()`.
- [ ] I know how `andThen()` works.

---

## Supplier

- [ ] I know when to use `Supplier`.
- [ ] I understand `get()`.
- [ ] I know where `Supplier` is commonly used.

---

# Specialized Functional Interfaces

- [ ] I know when to use `UnaryOperator`.
- [ ] I know when to use `BinaryOperator`.
- [ ] I know when to use `BiPredicate`.
- [ ] I know when to use `BiFunction`.
- [ ] I know when to use `BiConsumer`.

---

# JDK Usage

- [ ] I know why `Stream.filter()` uses `Predicate`.
- [ ] I know why `Stream.map()` uses `Function`.
- [ ] I know why `Stream.forEach()` uses `Consumer`.
- [ ] I know why `Stream.generate()` uses `Supplier`.
- [ ] I know why `Stream.reduce()` uses `BinaryOperator`.
- [ ] I know why `Map.forEach()` uses `BiConsumer`.

---

# Practical Understanding

- [ ] I can choose the correct Functional Interface for a given problem.
- [ ] I can replace an Anonymous Inner Class with a Lambda Expression.
- [ ] I know when to use Method References instead of Lambdas.
- [ ] I understand how Lambda Expressions are converted into Functional Interface implementations.
- [ ] I understand how APIs like `filter()` and `map()` execute the supplied Lambda Expressions.

---

# Interview Readiness

- [ ] I can explain Functional Interfaces without memorizing definitions.
- [ ] I can explain the relationship between Lambdas and Functional Interfaces.
- [ ] I can solve interview coding problems using Functional Interfaces.
- [x] I understand the difference between all core and specialized Functional Interfaces.
- [ ] I can explain real-world use cases for each Functional Interface.

---

# Score Yourself

| Score        | Readiness                                    |
| ------------:| -------------------------------------------- |
| **30–35**    | ⭐⭐⭐⭐⭐ Interview Ready                        |
| **24–29**    | ⭐⭐⭐⭐ Strong Understanding                    |
| **18–23**    | ⭐⭐⭐ Good, Needs Revision                     |
| **Below 18** | ⭐⭐ Revisit the Functional Interfaces section |

---

# Final Takeaway

> **If you can confidently explain when to use each Functional Interface and solve common interview problems without looking at the notes, you've mastered one of the most important Java 8 topics.**
