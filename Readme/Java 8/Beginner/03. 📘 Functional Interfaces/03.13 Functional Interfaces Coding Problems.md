# Interview Tip

Don't try to memorize every Functional Interface.

For each problem, first ask yourself:

- Do I need to **check** something?
- Do I need to **transform** something?
- Do I need to **perform** an action?
- Do I need to **generate** something?
- Do I need **one input** or **two inputs**?

Once you answer these questions, choosing the correct Functional Interface becomes straightforward.
