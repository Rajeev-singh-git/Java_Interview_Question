# 3.14 Functional Interfaces Interview Questions

This chapter contains interview questions covering all Functional Interfaces introduced in Java 8.

Use these questions to test your understanding before technical interviews.

---

# Beginner

1. What is a Functional Interface?
2. Why were Functional Interfaces introduced?
3. What is a Single Abstract Method (SAM)?
4. Can a Functional Interface have default methods?
5. Can a Functional Interface have static methods?
6. Is `@FunctionalInterface` mandatory?
7. Why do Lambda Expressions require Functional Interfaces?
8. Can an interface with two abstract methods be a Functional Interface?
9. What is the difference between a Functional Interface and a Lambda Expression?
10. What is meant by "Passing Behavior as a Parameter"?

---

# Intermediate

11. What is Predicate?
12. When should Predicate be used?
13. What is Function?
14. When should Function be used?
15. What is Consumer?
16. When should Consumer be used?
17. What is Supplier?
18. When should Supplier be used?
19. Explain UnaryOperator.
20. Explain BinaryOperator.
21. Explain BiPredicate.
22. Explain BiFunction.
23. Explain BiConsumer.
24. What is the difference between Function and UnaryOperator?
25. What is the difference between BiFunction and BinaryOperator?

---

# Advanced

26. How does a Lambda Expression become a Predicate?
27. Does Java pass methods as parameters?
28. How does `Stream.filter()` use Predicate?
29. How does `Stream.map()` use Function?
30. How does `Stream.forEach()` use Consumer?
31. How does `Stream.generate()` use Supplier?
32. How does `Stream.reduce()` use BinaryOperator?
33. Why does `Map.forEach()` use BiConsumer?
34. Explain `compose()` and `andThen()`.
35. Explain `Predicate.and()`.
36. Explain `Predicate.or()`.
37. Explain `Predicate.negate()`.
38. Explain `Function.identity()`.
39. When should custom Functional Interfaces be created?
40. Which Functional Interface would you choose for different scenarios?
