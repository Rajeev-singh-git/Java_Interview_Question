# 📘 Functional Interfaces Handbook

Functional Interfaces are the foundation of **Functional Programming in Java**.

A **Lambda Expression** by itself cannot exist—it always targets a **Functional Interface**. Modern Java features such as **Lambda Expressions**, **Method References**, **Streams**, **Optional**, and **CompletableFuture** all rely on Functional Interfaces.

This handbook helps you understand:

* What Functional Interfaces are
* Why they were introduced
* Java's built-in Functional Interfaces
* When to use each Functional Interface
* Interview-oriented coding patterns and questions

## 💡 Decision Guide

Need to...

| Requirement                                | Functional Interface |
| ------------------------------------------ | -------------------- |
| Perform a condition check                  | Predicate            |
| Transform one value into another           | Function             |
| Consume a value without returning anything | Consumer             |
| Generate a value without taking input      | Supplier             |
| Transform a value into the same type       | UnaryOperator        |
| Combine two values of the same type        | BinaryOperator       |
| Perform a condition check on two inputs    | BiPredicate          |
| Transform two inputs into one result       | BiFunction           |
| Consume two input values                   | BiConsumer           |
