# 🌊 Stream API Handbook

The **Stream API** is one of the most significant features introduced in **Java 8**, enabling developers to process collections in a **declarative**, **functional**, and **readable** manner.

Instead of writing explicit loops, Stream API allows you to describe **what** you want to achieve rather than **how** to achieve it.

This handbook covers the complete Stream API—from understanding what a Stream is to performing complex data transformations using stream operations and collectors.

# 💡 Stream Pipeline

```java
Collection / Array
        │
        ▼
 Create Stream
        │
        ▼
Intermediate Operations
(filter, map, sorted...)
        │
        ▼
Terminal Operation
(collect, count, reduce...)
        │
        ▼
      Result
```

---

# 💡 Common Stream Operations

| Requirement                | Operation                                 |
| -------------------------- | ----------------------------------------- |
| Filter elements            | `filter()`                                |
| Transform elements         | `map()`                                   |
| Flatten nested collections | `flatMap()`                               |
| Remove duplicates          | `distinct()`                              |
| Sort elements              | `sorted()`                                |
| Limit results              | `limit()`                                 |
| Skip elements              | `skip()`                                  |
| Peek elements              | `peek()`                                  |
| Iterate over elements      | `forEach()`                               |
| Convert to collection      | `collect()`                               |
| Aggregate values           | `reduce()`                                |
| Count elements             | `count()`                                 |
| Find first element         | `findFirst()`                             |
| Find any element           | `findAny()`                               |
| Match elements             | `anyMatch()`, `allMatch()`, `noneMatch()` |

---

# 💡 Common Collectors

| Requirement        | Collector          |
| ------------------ | ------------------ |
| Convert to List    | `toList()`         |
| Convert to Set     | `toSet()`          |
| Convert to Map     | `toMap()`          |
| Group elements     | `groupingBy()`     |
| Partition elements | `partitioningBy()` |
| Join Strings       | `joining()`        |
| Count elements     | `counting()`       |
| Calculate Average  | `averagingInt()`   |
| Calculate Sum      | `summingInt()`     |
| Get Statistics     | `summarizingInt()` |
