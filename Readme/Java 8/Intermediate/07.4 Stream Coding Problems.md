# 07.4 Stream Coding Problems

These problems are designed to test your understanding of the **Stream API**, **Stream Operations**, and **Collectors**.

Focus on writing **clean, readable, and idiomatic Java 8 code**.

---

# Easy

## 1. Filter Even Numbers

Given:

```java
List<Integer> numbers =
        List.of(2, 5, 8, 11, 14);
```

Print only the even numbers.

**Constraint**

Use the Stream API.

**Concepts Covered**

- `filter()`

- `forEach()`

---

## 2. Convert Names to Uppercase

Given:

```java
List<String> names =
        List.of("java", "spring", "docker");
```

Convert all names to uppercase.

**Constraint**

Use a Method Reference wherever possible.

**Concepts Covered**

- `map()`

- Method Reference

---

## 3. Remove Duplicate Elements

Given:

```java
List<Integer> numbers =
        List.of(2, 5, 2, 8, 5, 11);
```

Return a list containing only unique numbers.

**Concepts Covered**

- `distinct()`

- `collect()`

---

# Medium

## 4. Find the Highest Salary

Given a list of employees, find the employee with the highest salary.

**Constraint**

Do not use loops.

**Concepts Covered**

- `max()`

- Comparator

---

## 5. Group Employees by Department

Given a list of employees, group them by department.

Expected Result

```text
IT      → Employees

HR      → Employees

Finance → Employees
```

**Concepts Covered**

- `groupingBy()`

---

## 6. Find Employees Earning More Than ₹50,000

Return the names of employees whose salary is greater than ₹50,000.

Expected Result

```text
[Amit, Neha]
```

**Concepts Covered**

- `filter()`

- `map()`

- `toList()`

---

## 7. Count Employees in Each Department

Expected Result

```text
IT      → 5

HR      → 2

Finance → 3
```

**Concepts Covered**

- `groupingBy()`

- `counting()`

---

## 8. Create a Department-wise Employee Name List

Expected Result

```text
IT      → [Raj, Amit]

HR      → [Neha]
```

**Constraint**

Do not store complete Employee objects.

**Concepts Covered**

- `groupingBy()`

- `mapping()`

---

# Advanced

## 9. Find the Top 3 Highest Paid Employees

Return only the three employees with the highest salaries.

**Concepts Covered**

- `sorted()`

- `limit()`

---

## 10. Generate Department Salary Statistics

For every department, calculate:

- Total Salary

- Average Salary

- Minimum Salary

- Maximum Salary

- Employee Count

**Concepts Covered**

- `groupingBy()`

- `summarizingInt()`

---

## 11. Process Nested Orders

Suppose an `Order` contains multiple `OrderItem`s.

Return a list of all products ordered across every order.

**Concepts Covered**

- `flatMap()`

---

## 12. Mini Employee Analytics Challenge

Given a list of employees, implement the following using a single Stream pipeline wherever appropriate:

- Remove duplicate employees.

- Filter active employees.

- Convert names to uppercase.

- Group employees by department.

- Calculate salary statistics for each department.

- Print the names of the top three highest-paid employees.

**Constraint**

Use the Stream API and Collectors only.

**Concepts Covered**

- `filter()`

- `map()`

- `distinct()`

- `sorted()`

- `limit()`

- `groupingBy()`

- `summarizingInt()`

- `Collectors`

- Stream Pipeline
